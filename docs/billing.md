# Billing

## Virtual server hosting (OpenStack Cloud)

| What    | Unit       | TalTech internal | External  |
|---------|------------|------------------|-----------|
| CPU     | CPU*hour   | 0.002 EUR        | 0.003 EUR |
| Memory  | RAM*hour   | 0.001 EUR        | 0.0013 EUR|
| Storage | TB*year    | 20 EUR           | 80 EUR    |
